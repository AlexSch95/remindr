# Remindr

A lightweight desktop reminder application built with [Electron](https://www.electronjs.org/) for Windows. Remindr lives in your system tray and lets you create reminders with quick time presets or custom dates. A global hotkey opens an overlay for rapid reminder creation without interrupting your workflow.

![Electron](https://img.shields.io/badge/Electron-28.0-47848F?logo=electron&logoColor=white)
![License](https://img.shields.io/badge/License-MIT-green)

## Features

- **System Tray Integration** — Runs quietly in the background with quick access from the tray icon
- **Quick-Add Overlay** — Global hotkey (default: `Ctrl+Shift+R`) summons a floating input for rapid reminder creation
- **Quick Time Presets** — One-click buttons for 15m, 30m, 1h, 3h, and 24h
- **Persistent Notifications** — Reminders repeat every minute until marked complete or snoozed
- **Autostart Support** — Optional launch at system startup

## Techniques

This codebase uses several patterns that may be useful as reference implementations:

- **[Web Audio API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API)** generates notification sounds programmatically using oscillators and gain nodes—no external audio files needed
- **[CSS Custom Properties](https://developer.mozilla.org/en-US/docs/Web/CSS/Using_CSS_custom_properties)** power the theming system, with all colors defined in `:root` for easy modification
- **[CSS Gradients as Text Fill](https://developer.mozilla.org/en-US/docs/Web/CSS/gradient)** using `background-clip: text` creates the accent-colored icon effects
- **[CSS Keyframe Animations](https://developer.mozilla.org/en-US/docs/Web/CSS/@keyframes)** handle UI transitions including fade-in, pulse-glow, and floating effects
- **[Cubic-bezier Timing Functions](https://developer.mozilla.org/en-US/docs/Web/CSS/easing-function)** provide smooth, natural-feeling animation curves
- **[Custom Scrollbar Styling](https://developer.mozilla.org/en-US/docs/Web/CSS/::-webkit-scrollbar)** via `::-webkit-scrollbar` pseudo-elements matches the dark theme
- **XSS Prevention** — User input is escaped before rendering using a DOM-based sanitization pattern
- **Frameless Window** with custom title bar using Electron's [`frame: false`](https://www.electronjs.org/docs/latest/api/browser-window#new-browserwindowoptions) and [`-webkit-app-region: drag`](https://developer.mozilla.org/en-US/docs/Web/CSS/-webkit-app-region)

## Project Structure

├── assets/

- **[assets/](assets/)** — Application icons (`.ico` for Windows)
